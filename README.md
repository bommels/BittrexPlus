# BittrexPlus (Beta)
Chrome extension for Bittrex.com. This is still WiP and an official beta release will follow soon.

## Features
Supported currencies: USD, AUD, EUR, GBP, CAD, HKD, JPY and <a href="https://api.fixer.io/latest?base=USD">more</a>.
